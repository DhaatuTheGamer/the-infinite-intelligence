# Product Guidelines

## UI/UX Principles
- **Futuristic Minimalist**: Dark theme with high-contrast accents (Emerald, Rose, Indigo).
- **Progressive Disclosure**: Show agent "thinking" states only when relevant; keep the main synthesis central.
- **Visual Feedback**: Real-time status indicators (Thinking, Critiquing, Completed).

## Engineering Principles
- **Surgical Updates**: Modify only what is necessary to minimize regressions.
- **Type Rigor**: Maintain 100% TypeScript coverage for the orchestration engine.
- **Idempotency**: Ensure service calls can be retried safely.
