# Tech Stack

## Frontend
- **React 18**: UI component architecture.
- **TypeScript**: Type safety across orchestration state.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: Utility-first styling for the dark, futuristic theme.
- **Framer Motion**: Smooth animations for state transitions and graphs.
- **Lucide React**: Iconography system.

## AI & Orchestration
- **Google Generative AI SDK**: Direct integration with Gemini models.
- **Gemini 3.1 Pro**: High-reasoning tasks (Analysis, Assembly, Synthesis).
- **Gemini 3 Flash**: High-speed, low-cost tasks (Critique, Artifact Extraction).

## Infrastructure & Tools
- **Vitest**: Unit testing for service logic and component rendering.
- **jsPDF & html2canvas**: Executive report generation and export.
- **React Markdown**: Rendering of complex agent outputs.
